# **Sun**

***Efecto visual de sol con emisión brillante y partículas energéticas. Ideal para escenas espaciales o entornos místicos.***

</header>

## Preview

![Preview](./preview.gif)

## Requirements

* Unity: Unity 6
* Render Pipeline: Universal Render Pipeline

## Usage

Import the effect into your Unity project and place the prefab in the scene.

Adjust the scale, emission intensity, particle settings, and color depending on the visual style of the scene.
